# GitHub Trending RSS

This is a [Next.js](https://nextjs.org/) application that exposes an API endpoint to generate RSS feeds for [GitHub Trending](https://github.com/trending). It can be deployed directly to [Vercel](https://vercel.com/).

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000/api/rss?language=javascript` to retrieve an RSS feed of JavaScript trending repositories.
